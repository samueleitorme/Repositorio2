Pizza vegetal

9.99€

Una pizza vegetal es una deliciosa opción vegetariana para aquellos que buscan una comida rica en verduras frescas. Está hecha con una base de masa suave y crujiente, cubierta con salsa de tomate y queso mozzarella. Luego, se agrega una amplia variedad de verduras, como pimientos, cebolla, champiñones, aceitunas y espinacas, para crear una experiencia de sabor única. ¡Un bocado perfecto para cualquier amante de las verduras!
